# METAR & TAF Abbreviations

Decoder reference of **METAR and TAF abbreviations** used in aviation weather reports — precipitation, cloud cover, obscurations and report codes (e.g. `BKN` = broken cloud, `BR` = mist, `CAVOK` = ceiling and visibility OK).

61 codes in JSON and CSV, for weather decoders, flight-planning tools and aviation apps.

## Files
- [`metar-taf-codes.json`](metar-taf-codes.json) · [`metar-taf-codes.csv`](metar-taf-codes.csv)

## Example
```json
{ "code": "BKN", "meaning": "Broken (5-7 oktas)" }
```
